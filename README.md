# machine-learning-projects

Omar Ahmad · University of Washington
Two applied ML projects built in Python using scikit-learn and pandas.

EdX Student Certification Predictor
Predicts whether a student will earn course certification using logistic regression and random forest models. Includes data preprocessing, model comparison, hyperparameter tuning, and performance visualization.
→ EdX_Certification_Predictor.ipynb

COVID-19 Twitter Topic Modeling
Uses TF-IDF and Non-negative Matrix Factorization (NMF) to discover discussion topics across 100,000+ pandemic-era tweets. Includes 3D topic space visualization and outlier detection.
→ COVID19_Twitter_Topic_Modeling.ipynb
